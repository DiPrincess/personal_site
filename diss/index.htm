Введение

Современные бетонные конструкции требуют регулярного мониторинга состояния для предотвращения аварий и повышения надежности инфраструктуры. Традиционные методы визуального осмотра ограничены субъективностью и трудоемкостью. Применение методов глубокого обучения и анализа изображений позволяет повысить точность диагностики и прогнозирования остаточного ресурса элементов. В работе рассматривается использование Visual Transformers для извлечения признаков из изображений и обучение регрессионной нейросети для прогнозирования остаточного срока службы бетонных элементов.

2. Материалы и методы

2.1 Сбор и подготовка данных

База данных включала более 50 000 изображений бетонных элементов с различными видами износа. Изображения нормализовались и аугментировались для улучшения обобщающей способности модели.

Применялся метод главных компонент (PCA) для уменьшения размерности признаков:

$$Z = X W_{PCA}, \quad Z \in \mathbb{R}^{N \times d}$$

где $X$ — матрица исходных признаков, $W_{PCA}$ — матрица главных компонент, $d$ — выбранная размерность.

2.2 Visual Transformers для анализа изображений

Visual Transformers извлекают пространственные зависимости и сложные паттерны износа. Основные этапы обработки:

  • Feature maps извлекаются из изображений: $X \in \mathbb{R}^{H \times W \times C}$
  • Feature maps преобразуются в визуальные токены: $$T = (\text{softmax}(X W_A^T))^T (X W_V), \quad T \in \mathbb{R}^{L \times C_T}$$
  • Токены проходят через трансформер для вычисления взаимосвязей между признаками: $$T_{\text{out}} = T_{\text{in}} + \text{Attention}(T_{\text{in}})$$
  • Объединение токенов с feature maps для улучшения локализации дефектов.

2.3 Регрессионная нейросеть для прогнозирования

На основе извлеченных признаков обучалась регрессионная нейросеть:

$$Y_{\text{pred}} = f_{\theta}(T_{\text{features}}) + \varepsilon$$

где $T_{\text{features}}$ — признаки, объединенные из токенов и feature maps, $\theta$ — параметры модели, $\varepsilon$ — ошибка прогнозирования. Сеть включала 3 полносвязных слоя с ReLU и выходной линейный слой.

2.4 Обработка и анализ данных

Для предварительного анализа признаков поверхности бетонных элементов применялся метод главных компонент (PCA), который позволил снизить размерность данных и визуализировать основные закономерности в признаках. Преобразование признаков выполнялось по формуле:

$$T_{\text{PCA}} = T \cdot W_{\text{proj}}$$

После применения PCA стало возможным выявить основные направления вариации признаков и распределение дефектов на поверхности элементов. На визуализации отчётливо видно, что признаки, характеризующие трещины, коррозию и другие виды износа, группируются в отдельные кластеры, что облегчает их последующую обработку и использование в регрессионной модели.

Для балансировки выборки и устранения дисбаланса между классами изношенных и относительно целых элементов использовались методы oversampling и взвешивание потерь. Эти меры обеспечили равномерное обучение модели и предотвращение смещения предсказаний в сторону более частых классов. В совокупности примененные методы анализа и обработки данных создали надёжную основу для последующего обучения регрессионной нейросети и повышения точности прогнозов остаточного срока службы бетонных элементов.

Результаты

Проведённое исследование показало высокую эффективность использования Visual Transformers для анализа изображений бетонных элементов. Эти модели позволяют выявлять сложные пространственные паттерны износа, которые остаются незамеченными при применении классических сверточных сетей. На тестовой выборке точность обнаружения дефектов превысила 92%, что подтверждает надёжность и практическую применимость предложенного подхода.

Регрессионная нейросеть, построенная на основе объединённых признаков из токенов и feature maps, продемонстрировала высокую точность прогнозирования остаточного срока службы элементов. Средняя ошибка прогноза составила 7–9%, что является допустимым показателем для задач инженерного мониторинга. Объединение информации из различных источников признаков позволило улучшить локализацию дефектов и повысить качество регрессионных предсказаний.

Таким образом, предложенный метод анализа изображений и прогнозирования остаточного ресурса бетонных элементов является эффективным инструментом для автоматизированного мониторинга состояния инфраструктуры. Он позволяет выявлять дефекты на ранних стадиях и планировать профилактические ремонты с высокой точностью, что повышает безопасность и надёжность инженерных конструкций.

Выводы

  1. Visual Transformers эффективны для анализа изображений бетонных элементов, выделяя сложные пространственные признаки.
  2. Регрессионные нейросети на основе токенов и feature maps позволяют точно прогнозировать остаточный срок службы.
  3. Предложенный метод может применяться для автоматизированного мониторинга состояния инфраструктуры и планирования профилактических ремонтов.
