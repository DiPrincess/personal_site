Title: Assessment of Wear and Lifetime Prediction of Concrete Infrastructure Elements Based on Image Analysis and Regression Neural Network Methods

Introduction

Modern concrete structures require regular condition monitoring to prevent failures and improve infrastructure reliability. Traditional visual inspection methods are limited by subjectivity and labor intensity. Applying deep learning and image analysis methods improves the accuracy of diagnostics and prediction of the remaining service life of elements. This study examines the use of Visual Transformers to extract features from images and train a regression neural network to predict the remaining service life of concrete elements.

2. Materials and Methods

2.1 Data Collection and Preparation

The database included over 50,000 images of concrete elements with various types of wear. Images were normalized and augmented to improve model generalization.

Principal Component Analysis (PCA) was applied to reduce feature dimensionality:

$$Z = X W_{PCA}, \quad Z \in \mathbb{R}^{N \times d}$$

where $X$ is the original feature matrix, $W_{PCA}$ is the principal component matrix, and $d$ is the chosen dimension.

2.2 Visual Transformers for Image Analysis

Visual Transformers extract spatial dependencies and complex wear patterns. The main processing steps:

  • Feature maps are extracted from images: $X \in \mathbb{R}^{H \times W \times C}$
  • Feature maps are converted into visual tokens: $$T = (\text{softmax}(X W_A^T))^T (X W_V), \quad T \in \mathbb{R}^{L \times C_T}$$
  • Tokens pass through the transformer to compute relationships between features: $$T_{\text{out}} = T_{\text{in}} + \text{Attention}(T_{\text{in}})$$
  • Token aggregation with feature maps improves defect localization.

2.3 Regression Neural Network for Prediction

A regression neural network was trained on the extracted features:

$$Y_{\text{pred}} = f_{\theta}(T_{\text{features}}) + \varepsilon$$

where $T_{\text{features}}$ are features aggregated from tokens and feature maps, $\theta$ are model parameters, and $\varepsilon$ is prediction error. The network consisted of three fully connected layers with ReLU and a linear output layer.

2.4 Data Processing and Analysis

PCA was applied for preliminary analysis of surface features of concrete elements, reducing dimensionality and visualizing main patterns:

$$T_{\text{PCA}} = T \cdot W_{\text{proj}}$$

PCA allowed identification of major feature variations and clustering of defects such as cracks and corrosion, facilitating subsequent regression modeling.

Oversampling and loss weighting were applied to balance the dataset and reduce class imbalance, ensuring uniform model training and preventing prediction bias. These preprocessing steps provided a reliable foundation for training the regression neural network and improving prediction accuracy.

Results

The study demonstrated high efficiency of Visual Transformers for analyzing concrete element images. These models detect complex spatial wear patterns missed by classical convolutional networks. Defect detection accuracy exceeded 92% on the test set, confirming reliability and practical applicability.

The regression neural network built on combined token and feature map information achieved a prediction error of 7–9%, which is acceptable for engineering monitoring tasks. Integrating multiple feature sources improved defect localization and prediction quality.

Thus, the proposed image analysis and service life prediction method is an effective tool for automated infrastructure monitoring, enabling early defect detection and precise maintenance planning, increasing safety and reliability of structures.

Conclusions

  1. Visual Transformers are effective for concrete image analysis, highlighting complex spatial features.
  2. Regression neural networks using tokens and feature maps allow accurate prediction of remaining service life.
  3. The proposed method can be applied for automated infrastructure monitoring and preventive maintenance planning.
