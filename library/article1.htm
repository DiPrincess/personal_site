Аннотация

В статье выявляется потенциал нейронных сетей при решении задач распознавания в области гистологии и их способности улучшать качество медицинской диагностики. Выполняется анализ качества распознавания изображений на трех нейросетевых моделях.

Введение

С развитием технологий в области искусственного интеллекта, нейронные сети становятся мощным инструментом для автоматизации процессов анализа изображений, включая гистологические. Сверточные нейронные сети (CNN) продемонстрировали выдающиеся результаты в задачах классификации и распознавания изображений, что открывает новые горизонты для их применения в медицинской диагностике [1]. Использование нейронных сетей для распознавания гистологии предлагает возможность повышения точности и скорости диагностики, а также снижение нагрузки на врачей-диагностов.

Определим наиболее эффективные параметры нейронной сети для решения задач гистологии. Для этого экспериментальным путем подберем архитектуру нейросети на гистологическом датасете.

Основная часть

Одной из наиболее успешных архитектур нейронных сетей для обработки изображений является ResNet, которая была разработана для решения проблем, связанных с обучением глубоких нейронных сетей. В исследовании предлагается использовать эту нейронную сеть для классификации гистологических тканей. Для обучения сети и оценки классификации использованы два набора данных, доступных через платформу Kaggle. Один набор данных CRC (Colorectal cancer) содержит 7000 снимков тканей колоректального рака, окрашенных гематоксимином и эозином. Эти данные были использованы для валидации модели.

Другой набор данных NCT (National Center For Tumor Diseases), включающий 100 000 гистологических изображений тканей колоректального рака, использован для обучения модели. Для оптимизации гиперпараметров применены функции библиотеки Optuna, которая предоставляет эффективные методы поиска оптимальных значений гиперпараметров с использованием байесовской оптимизации.

В проекте применен фреймворк PyTorch, позволяющий создавать и обучать нейронные сети, обеспечивая гибкость и высокую производительность, и библиотека Torchvision – для работы с изображениями.

Перед обучением необходимо подготовить данные. Важно, чтобы наборы данных содержали сбалансированное количество изображений для каждого класса. Результаты визуализации представлены на рисунке 2.

Данные из выборки, включающей 100 000 изображений тканей колоректального рака, распределены равномерно, нет сильных выбросов. Данные CRC-VAL-HE-7k распределены менее равномерно, что важно учитывать при балансировке классов с помощью oversampling. Результат применения oversampling представлен на рисунке 3.

Для преобразования гистологических изображений в векторные представления реализована функция feature_extraction, использующая предобученную модель ResNet-50 для извлечения признаков из изображений. Этот подход позволяет эффективно преобразовать изображения в векторные представления для обучения классификатора. Для визуализации структуры данных применен метод главных компонент (PCA), который уменьшает размерность данных, сохраняя вариативность.

Далее проведена оценка трех моделей классификации: AdaBoost, RandomForest и MLP. Результаты: cv = 0.505, test = 0.474; cv = 0.726, test = 0.663; cv = 0.848, test = 0.828. Модель MLPClassifier показала наилучшую производительность, RandomForest уступила MLP по точности, AdaBoostClassifier – наименее точная.

Выводы

Для повышения производительности модели MLPClassifier проведена оптимизация гиперпараметров с использованием Optuna. Наилучшая точность составила 0,862. Искусственный интеллект и нейронные сети открывают новые горизонты в области гистологии и медицинской диагностики. Использование Residual Network (ResNet) позволяет глубоко понять потенциал современных технологий при анализе гистологических изображений.
